🏕️ Деятельность

  • Возрастная категория детей: 7–17 лет
  • Даты проведения смен на календарный год: с 1 июня 2026 г. (летняя оздоровительная кампания)
  • Реализуемые дополнительные образовательные программы: В рамках воспитательной работы (программа прилагается в разделе "Документы")
  • Дополнительные оздоровительные программы: Организовано 2-разовое питание, оздоровительные мероприятия по плану лагеря дневного пребывания
  • Методические разработки: Имеются планы воспитательной работы, сценарии мероприятий

🏢 Материально-техническое обеспечение и оснащенность

  • Дата ввода в эксплуатацию объектов: 1970 год
  • Дата последнего капитального ремонта: 2014 год
  • Условия проживания детей: Отсутствуют (лагерь с дневным пребыванием, проживание не предусмотрено)
  • Условия питания детей: 2-разовое меню, соответствующее всем нормам СанПиН
  • Наличие оборудованных учебных кабинетов: Нет
  • Наличие объектов для практических занятий: Нет
  • Наличие библиотек: Есть
  • Наличие объектов спорта: Нет
  • Материально-техническое обеспечение территории для оздоровительной деятельности: Игровая площадка, столовая, помещения для отрядов

💰 Услуги, предоставляемые организацией

  • Порядок оказания платных услуг: Платные услуги не оказываются. Деятельность осуществляется в рамках государственного задания.
  • Средняя стоимость одного дня пребывания: Бесплатно (бюджетное финансирование)
  • Сведения о компенсации стоимости услуг: Не предусмотрена ввиду бесплатности отдыха
  • Перечень документов ребенка для зачисления: Заявление, копия свидетельства о рождении, копия паспорта законного представителя, СНИЛС ребенка, медицинская справка.
  • Перечень одежды, обуви и гигиенических принадлежностей: Сменная обувь, полотенце, средства личной гигиены (мыло, зубная щетка, паста).
